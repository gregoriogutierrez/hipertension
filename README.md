MI PRESIÓN ARTERIAL

Aplicación web para el registro y seguimiento de la presión arterial y la frecuencia cardiaca.

Descripción

MI PRESIÓN ARTERIAL es una aplicación HTML que permite registrar mediciones de presión arterial (sistólica y diastólica) y pulso, almacenándolas localmente en el dispositivo del usuario para su posterior consulta y análisis.

La aplicación genera estadísticas por periodos (semanales, mensuales, trimestrales y anuales) y calcula promedios diarios por franjas horarias.

Protección de datos y privacidad

Compromiso de privacidad

Esta aplicación ha sido diseñada con un enfoque de máxima privacidad para el usuario.

* Todos los datos se almacenan exclusivamente en el navegador web mediante la tecnología Local Storage.
* No existe ningún servidor asociado a la aplicación.
* No se realiza ninguna transmisión de datos a Internet.
* No se recopilan datos personales.
* No se utilizan cookies de seguimiento.
* No se utilizan herramientas de analítica ni servicios de terceros.
* Ninguna información médica o sanitaria sale del dispositivo del usuario.

Por tanto:

Todas las mediciones permanecen únicamente en el dispositivo donde se introducen y son gestionadas exclusivamente por el propio usuario.

Responsabilidad del usuario

Los datos almacenados dependen del navegador utilizado. Si el usuario borra los datos de navegación, la caché o el almacenamiento local del navegador, las mediciones registradas pueden perderse.

Se recomienda realizar copias de seguridad periódicas si se considera necesario.

Uso previsto

Esta aplicación está destinada únicamente al registro y seguimiento personal de datos de presión arterial y frecuencia cardiaca.

No sustituye el diagnóstico, tratamiento o seguimiento realizado por profesionales sanitarios.

Ante cualquier duda médica, el usuario debe consultar con su médico o profesional sanitario de referencia.
