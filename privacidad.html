MI PRESIÓN ARTERIAL

Aplicación web para registrar y consultar mediciones de presión arterial y frecuencia cardiaca desde el propio dispositivo del usuario.

Resumen de privacidad

Esta aplicación está diseñada para funcionar de forma local. Las mediciones se guardan en el navegador del usuario y no se envían a ningún servidor.

Tus datos permanecen en tu dispositivo. Ninguna medición médica sale del navegador.
Uso previsto Registro personal de presión arterial, pulso y consulta de estadísticas por periodos y franjas horarias.

Descripción

MI PRESIÓN ARTERIAL permite registrar mediciones de presión arterial sistólica, presión arterial diastólica y pulso.

La aplicación genera estadísticas semanales, mensuales, trimestrales y anuales, y calcula promedios diarios por franjas horarias: mañana, tarde y noche.

Protección de datos y privacidad

La aplicación se ha desarrollado con un enfoque de máxima privacidad para el usuario.

  • ✓
    Los datos se almacenan exclusivamente en el navegador mediante Local Storage.
  • ✓
    No existe ningún servidor asociado a la aplicación.
  • ✓
    No se realiza ninguna transmisión de datos a Internet.
  • ✓
    No se recopilan datos personales.
  • ✓
    No se utilizan cookies de seguimiento.
  • ✓
    No se usan herramientas de analítica ni servicios de terceros.
  • ✓
    Ninguna información médica o sanitaria sale del dispositivo del usuario.

Conclusión

Todas las mediciones permanecen únicamente en el dispositivo donde se introducen y son gestionadas exclusivamente por el propio usuario.

Responsabilidad del usuario

Los datos quedan guardados en el navegador del usuario. La aplicación no dispone actualmente de una función de copia de seguridad ni de exportación de datos, por lo que el usuario debe tener en cuenta que, si borra los datos del navegador o cambia de dispositivo, las mediciones pueden perderse.

Uso sanitario previsto

Esta aplicación está destinada únicamente al registro y seguimiento personal de datos de presión arterial y frecuencia cardiaca.

No sustituye el diagnóstico, tratamiento o seguimiento realizado por profesionales sanitarios.

Ante cualquier duda médica, el usuario debe consultar con su médico o profesional sanitario de referencia.
